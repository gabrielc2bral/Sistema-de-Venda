# Requisitos

## 1. Cadastro

- O sistema deve permitir o cadastro de usuários.
- O sistema deve permitir o cadastro de produtos.
- O sistema deve permitir o cadastro de pedidos.

## 2. Consulta
- O sistema deve permitir a busca de usuários cadastrados.
- O sistema deve permitir a busca de vendedores cadastrados.
- O sistema deve permitir a busca de produtos cadastrados.
- O sistema deve permitir a busca de pedidos por identificador.


## 3. Relatórios

- O sistema deve criar relátorio de vendas por vendedor (ativo).
- O relatorio deve apresentar o total de vendas e o valor ganho com cada venda.
- O Sistema deve criar relátorio de estoque de produtos.
